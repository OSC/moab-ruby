# Moab

Ruby wrapper for the Moab binaries. It uses the XML output of Moab for easy
parsing.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'moab'
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install moab
```
